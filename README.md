# PS2 Emulator Presentation

This repository contains the slides and resources for a school presentation about **PCSX2**, the open-source PlayStation 2 emulator.

---

## 📦 Downloading the PS2 BIOS (v2.30)

The PS2 BIOS files are required to run PCSX2. To download version 2.30:

1. Go to the BIOS file in this repository:
   👉 [PS2_Bios_v2.30.zip](https://github.com/VitorNoe/ps2emulatorpresentation/blob/main/PS2_Bios_v2.30.zip)
2. On that page, click **"View raw"** to start the download.
3. Once downloaded, extract the `.zip` file and copy the BIOS files into the `bios` folder inside your PCSX2 installation directory.

> **Note:** Any version of the PS2 BIOS works with PCSX2, but v2.30 is the most up-to-date one available here.

## 🕹️ Getting Started with PCSX2

| Step | What to do |
|------|-----------|
| 1 | Download PCSX2 from [pcsx2.net](https://pcsx2.net) — choose **Latest Nightly** |
| 2 | Download the BIOS files (see above) and place them in the `bios` folder |
| 3 | Download PS2 games (`.iso`) from [Myrient](https://myrient.erista.me/files/Redump/Sony%20-%20PlayStation%202/) |
| 4 | Open PCSX2, add your games folder under **Game List**, and play |

## ⚙️ System Requirements

**Minimum:**
- OS: Windows 10/11 or Linux
- CPU: x86-64 with SSE4.1 support (e.g. Intel Core i7-2760QM or AMD FX-8350)
- RAM: 8 GB
- GPU: Vulkan 1.1 / Direct3D 11 compatible (e.g. NVIDIA GeForce GT 710)

**Recommended:**
- CPU: Intel Core i7-4790 or AMD Ryzen 5 1600 (or better)
- GPU: NVIDIA GTX 1650 / AMD RX 570 or better
- RAM: 16 GB or more
